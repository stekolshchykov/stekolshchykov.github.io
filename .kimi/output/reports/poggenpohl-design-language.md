# Poggenpohl +SEGMENTO Design Language Study
## Creative reference for TINEO — a premium Irish custom furniture brand

**Reference URL:** https://www.poggenpohl.com/en/kitchen-architecture/products/segmento/  
**Date:** 2026-06-25

---

## 1. Summary of +SEGMENTO principles

+SEGMENTO is described by Poggenpohl as **“a classic of modern elegance”**. Its design language is built on a few tightly controlled ideas:

| Principle | What it means on the page |
|---|---|
| **Calm objectivity** | Long, straight horizontal lines, flush planes and a deliberate lack of visual noise. The product is treated as architecture rather than cabinetry. |
| **Handleless reduction** | Doors and drawers are opened via recessed grip profiles, touch-latch or push-to-open mechanisms. Hardware is invisible; the read is of continuous surfaces and fine shadow lines. |
| **Thin worktops / thin planes** | Worktops are slim (≈12 mm in many configurations), which emphasises lightness. Island snack sides can read thicker for sculptural contrast, but the working edge stays delicate. |
| **Meaningful detail** | Macro moments reward attention: 100 lightly chamfered stripes forming a recessed seating area, natural stone wrapping drawer fronts, bronze or aluminium frame profiles, hand-finished lacquer corners. |
| **Material as personality** | The same system can read “strictly purist or entirely opulent” depending on colour and material. Salt white glass ceramic reads minimal; smoked oak + Rosso Lepanto marble reads luxurious. |
| **Modularity with monolithic effect** | Units are modular and configurable, yet compositions read as single, continuous masses — floor-to-ceiling, wall-to-wall, island-as-sculpture. |
| **Timelessness over trend** | The vocabulary is neutral, geometric and proportion-led; colour accents come from stone or a single curated object. |

### Signature materials and finishes observed

- **Surfaces:** polar white and salt white glass ceramic, quartz, Dekton / ceramic, Taj Mahal / Calacatta / Rosso Lepanto marble, Marazzi porcelain.
- **Woods:** smoked oak, black matt brushed spruce, knotty spruce, natural core ash, teak lava, walnut.
- **Metals / frames:** bronze anodised aluminium, silver anodised aluminium, stainless steel worktops, brass accents.
- **Finishes:** super-matt lacquer, high-gloss lacquer, brushed satin, hand-finished stucco, fluted / reeded glass, mother-of-pearl.

## 2. Adaptation guide: from kitchen language to TINEO custom furniture

The +SEGMENTO vocabulary is fundamentally about **planes, joins and material continuity**. It transfers naturally to wardrobes, living rooms, studies and offices.

| +SEGMENTO concept | TINEO translation |
|---|---|
| Handleless kitchen cabinets | Handleless wardrobes, media units, sideboards and desks with recessed grip rails or push-to-open drawers. |
| Thin worktop | Thin desk tops, console surfaces, credenza tops — a slim “working plane” that appears to float. |
| Island monolith | Freestanding wardrobe island, media island, or office storage pod treated as a single sculptural volume. |
| Floor-to-ceiling cabinetry | Full-height built-in libraries, wardrobes and home-office joinery. |
| Chamfered / fluted seating detail | Fluted or reeded wardrobe doors, panelled bedheads, desk end panels. |
| Stone-framed drawers | Stone or timber wrapping the carcass of a piece — a “material loop” that unifies sides and top. |
| Integrated appliances | Integrated AV, cable management, lighting and charging — function concealed behind calm planes. |
| Material-led personality | Each room uses a single hero material (e.g. smoked oak, Carrara marble, soft clay plaster) plus one accent metal. |

### Tone-of-voice shift for TINEO

Poggenpohl speaks with German precision (“Made in Germany”, “exact to a tenth of a millimetre”). TINEO should speak with Irish warmth and craft: **“made to measure in Ireland”, “quiet by design”, “built to live with”.** Keep the same restraint, but replace industrial exactness with human materiality and place.

## 3. Recommended visual direction for TINEO

### 3.1 Colour palette

A calm, material-first palette anchored in stone and timber, with one warm metallic accent.

| Role | Suggested colour | Notes |
|---|---|---|
| Background / paper | `#F5F3EF` warm off-white | gallery-wall neutral; lets wood and stone breathe. |
| Primary surface | `#EAE6DC` marble cream | for large product surfaces and cards. |
| Mid-tone / secondary | `#B8B4AD` stone grey | type labels, dividers, hover states. |
| Dark accent | `#2C2C2C` deep graphite | footer, CTA panels, primary text. |
| Timber warmth | `#5A4636` smoked oak | use for image tints, wood swatches, warm accents. |
| Metallic accent | `#A68A64` brass / bronze | links, rules, hover underlines, small UI moments. |
| Botanical muted | `#8A9480` sage | optional living-space accent; keep desaturated. |

Avoid cold pure whites and saturated primary colours. Every colour should feel like it could exist in a stone, timber or plaster sample.

### 3.2 Typography mood

Take the Poggenpohl pairing — **editorial serif + technical sans** — and adapt it for TINEO.

- **Display / headlines:** A refined transitional or modern serif with quiet contrast. *Mood equivalents:* Tiempos (used by Poggenpohl), Freight Text, Canela, SangBleu, GT Sectra.
- **Body / UI / labels:** A neutral, slightly tight grotesk or neo-grotesk. *Mood equivalents:* Maison Neue (used by Poggenpohl), Neue Haas Grotesk, Söhne, Graphik, Inter.
- **Large display type:** Use the sans in very light weights at huge scale for section numbers, oversized category labels, or background typography.

Typographic rules:

- Headlines: generous line-height (1.1–1.2), medium to light weight.
- Labels / captions: uppercase or small-caps, wide letter-spacing, 11–12 px.
- Body: 16–18 px, line-height 1.5–1.6, max-width ~70 characters.
- Use type as a compositional line: left-aligned, ragged right; avoid centred paragraphs.

### 3.3 Photography style

- **Setting:** Real Irish interiors or considered architectural spaces — not white-studio cut-outs. Show pieces in rooms with natural light, stone floors, plaster walls, large windows.
- **Framing:** Wide shots establish calm volume; detail shots reveal joinery, grain, worktop edges, drawer internals.
- **Lighting:** Soft daylight with gentle shadow; avoid harsh flash. Let light rake across fluted or textured surfaces.
- **Props:** Extremely minimal — a ceramic vessel, a single branch, a book, a hand-thrown bowl. Props should look collected, not styled.
- **People:** Prefer absent or implied (a pulled-out chair, a half-open notebook). Keep focus on the object and the space.
- **Colour grade:** Warm, desaturated, slightly lifted blacks. Emphasise tactility.

### 3.4 Layout principles

- **Whitespace is a material.** Give every image and text block room to breathe; vertical padding should feel generous (80–160 px on desktop).
- **Asymmetric two-column rhythm.** Text block left / image right, or image bleeding to one edge with a small caption floating inside.
- **Edge bleed.** Let hero images and full-width project images touch the viewport edges; this removes the “boxed website” feeling.
- **Thin rules.** Use 1 px hairlines in stone grey to separate sections, not heavy dividers.
- **Small caps labels.** Mark sections with tiny uppercase labels (“Wardrobes · Dublin”, “Material · Smoked oak”).
- **Modular grid.** A 12-column grid with consistent gutters lets project cards and material cards align cleanly.

### 3.5 Motion principles

Premium furniture sites move slowly. Motion should feel like a considered gesture, not decoration.

- **Page load:** Large images fade in over 600–900 ms; type reveals slightly after.
- **Scroll:** Subtle parallax on hero images (0.05–0.1 speed). Sections fade up as they enter the viewport.
- **Hover on images:** Very gentle scale (1.02–1.04) over 600 ms; avoid cropping or dramatic zoom.
- **Links:** Underline grows from left to right, or a small arrow shifts 4–8 px.
- **Cards:** Minimal lift — no heavy shadows. A card might reveal its caption on hover with a soft opacity transition.
- **Cursor:** Default arrow; avoid gimmicky custom cursors that fight the calm mood.

## 4. What makes the reference feel “premium” and “wow”

1. **Image-first hierarchy.** The product dominates; navigation and chrome recede.
2. **Macro craft moments.** Close-ups of a worktop edge, a drawer interior, a fluted panel prove quality without saying it.
3. **Restraint.** Only one type of link style, one button style, one colour of rule. The system is visibly disciplined.
4. **Architectural context.** Kitchens are shown inside beautiful houses, not showrooms. TINEO should do the same for Irish homes.
5. **Heritage proof points.** Awards, “Made in Germany”, history — these are displayed as quiet rows, not boastful banners.
6. **Personalisation promise.** The word “your” is implied by the configurability shown: different materials, different moods, same DNA.
7. **Consistent terminology.** The “+” prefix, product naming and modular language create a coherent family.

## 5. Comparison with ultra-premium furniture / interior brands

| Brand | Digital / design signature | What TINEO can borrow |
|---|---|---|
| **Molteni&C / Dada** | Vincent Van Duysen’s quiet minimalism; travertine, walnut, bone greys; monolithic kitchen forms; large editorial photography; designer attribution. | Use architect/designer partnerships; material palette of stone + smoked timber + warm grey; full-height joinery systems. |
| **Poliform** | Modular grid layouts, video-rich product pages, calm colour scheme, seamless transitions from kitchen to wardrobe to living. | Apply a single modular system language across all TINEO categories (wardrobes / living / office). |
| **Boffi** | “History of design” narrative; sculptural Corian monoblocks; category portals (Kitchens / Bathrooms / Wardrobes); restrained Neue Grotesk typography. | Use a strong craft story; sculptural freestanding pieces; clear category entry points. |
| **Minotti** | Deep, warm colour schemes, rich lifestyle visuals, subtle micro-interactions, understated luxury. | Add warmth and tactile comfort to the palette; lifestyle photography with implied use. |
| **Flexform** | “Discreet elegance”, Morandi neutrals, deconstructed comfort, anti-bling luxury, timeless sofa archetypes. | Tone-of-voice: quiet, confident, never loud; focus on how pieces feel to live with. |

### Common patterns across all brands

- **Full-bleed architectural photography** as the dominant communication tool.
- **Neutral / earthy / stone-derived palettes**; colour comes from material, not decoration.
- **Serif + sans typography pairing** for editorial credibility.
- **Modular, grid-based layouts** that mirror the modularity of the products.
- **Designer / maker narrative** (craftsperson, architect, atelier).
- **Minimal chrome UI**; navigation is thin or hidden.
- **Slow, subtle motion** — fade-ins, gentle scale, parallax.
- **Showroom / flagship experience** sections that turn the website into a doorway to physical space.

## 6. Ten specific UI patterns to borrow / adapt

1. **Full-bleed category hero**  
   A near-full-viewport image of a TINEO piece in a real room. A small uppercase label at the top, a large serif headline at the bottom-left, and a barely-there scroll indicator.

2. **Asymmetric feature block**  
   One side holds a block of text and a hairline rule; the other side holds an image that bleeds to the viewport edge. The text block overlaps the image by a small margin for depth.

3. **Handleless product card**  
   Image-only card with no border. On hover, a thin caption slides up from the bottom: title in serif, category in small-caps sans.

4. **Material macro card**  
   Square or 4:5 crop showing extreme close-up of timber grain, marble vein or fluted detail. A small circular swatch and a two-word caption sit in the corner.

5. **Edge-to-edge configuration gallery**  
   A horizontal scroll or carousel of wide images showing the same furniture system configured for different rooms (wardrobe, study, living).

6. **Sticky chapter navigation**  
   On long product pages, a minimal left-hand or top-centre menu that anchors to sections: Overview · Details · Materials · Configurations · Projects.

7. **“Discover more” link treatment**  
   Text link with a thin underline that animates on hover; no button background. Keeps UI light and editorial.

8. **Awards / trust bar**  
   A single row of monochrome award logos and one-line craft claims, separated by hairlines. Use sparingly — one per page is enough.

9. **Split-panel CTA**  
   Left half: dark graphite panel with a large serif question (“Ready to design your space?”). Right half: detail image. The divide is a clean vertical line, not a box.

10. **Configurator / hotspot teaser**  
    A still hero image with subtle animated dots. Hovering a dot reveals a material name and finishes. This borrows from luxury product pages without building a full 3D tool.

## 7. Example treatments for TINEO

### 7.1 Hero section

```
[Full-bleed image: a floor-to-ceiling Irish study in warm plaster,
                  smoked-oak desk and shelving running wall-to-wall]

Top-left:     TINEO          [small caps, sans, 12px, letter-spaced]
Bottom-left:  Custom furniture,
              quiet by design  [large serif, 64–96px, warm off-white]
Bottom-right: Scroll          [thin vertical line + word, fades on scroll]
```

Rules:
- Text sits inside the image but never crowds the architecture.
- Use a subtle gradient scrim only if needed for legibility; prefer darker image areas behind type.
- Hero is 90–100 vh; no carousel autoplay.

### 7.2 Project card

```
┌─────────────────────────────────────┐
│                                     │
│    [Full-bleed project photograph]  │
│                                     │
│    Blackrock Residence              │
│    Wardrobes · 2026                 │
│    ─────────────                    │
│    View project  →                  │
│                                     │
└─────────────────────────────────────┘
```

Rules:
- Card has no border and no shadow.
- Metadata sits bottom-left with generous padding.
- Hover: image scales 1.03, “View project” underline animates.
- On mobile, metadata is always visible; no tap-to-reveal.

### 7.3 Material card

```
┌─────────────────┐
│                 │
│  [Macro image   │
│   of fluted     │
│   smoked oak]   │
│                 │
│  Smoked oak     │
│  hand-brushed   │
│            ●    │  ← small swatch circle
└─────────────────┘
```

Rules:
- Square aspect ratio; consistent across all materials.
- Caption uses sans small-caps + serif material name.
- Swatch circle uses the actual material texture, not flat colour.
- Hover: image lightens slightly, caption shifts up 8 px.

### 7.4 CTA treatment

```
┌──────────────────────┬──────────────────────┐
│                      │                      │
│  Ready to design     │  [Close-up image:    │
│  your space?         │   dovetail joint or  │
│                      │   workshop hand-     │
│  Book a private      │   finishing timber]  │
│  consultation.       │                      │
│                      │                      │
│  Book now  →         │                      │
│                      │                      │
└──────────────────────┴──────────────────────┘
```

Rules:
- Left panel: deep graphite background, warm off-white type.
- Headline in large serif; link in small-caps sans with arrow.
- Right panel: detail photography with natural light.
- On mobile, stack image above text panel.
